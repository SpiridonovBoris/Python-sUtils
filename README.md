# Python-s_utils
A collection of syntactic sugar functions for Python

# Install
Add the following line to your ~/.bashrc:
```bash
# S_utils
 if [ -f "${HOME}/Path/to/the/s_utils.py" ]; then
   source "${HOME}/Path/to/the/s_utils.py"
 fi
```

# Usage
## As a Library
After the shebang, add the following line at the beginning of your Python script:
```Python
import s_utils
```
